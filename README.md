# 強密碼產生器

## 概述

**強密碼產生器** 是一個跨平台的密碼管理工具，支援 Windows、macOS 與 Linux。它提供安全的密碼生成、儲存、管理和導出功能。應用啟動時需要通過系統級認證（如 Windows 密碼或 Unix PAM 認證），確保數據安全性。

## 主要功能

### 1. **密碼生成** 🔐
- 三個強度等級選擇：
  - **低強度**：8 字元（字母 + 數字）
  - **中強度**：10 字元（字母 + 數字 + 特殊符號）
  - **高強度**：12 字元（字母 + 數字 + 特殊符號）
- 生成密碼後自動計算強度評估，以進度條顯示（紅/黃/綠）
- 一鍵複製密碼到剪貼板

### 2. **密碼儲存** 💾
- 輸入服務名稱（如 'Gmail', 'GitHub'）、用戶名、密碼
- 密碼自動加密存儲在 SQLite 數據庫
- **數據目錄**：
  - **Windows**: `%LOCALAPPDATA%\.password_generator\`
  - **macOS/Linux**: `~/.password_generator/`

### 3. **密碼管理** 📋
- **查看**：點擊查看按鈕查看已儲存密碼詳情
- **編輯**：修改已儲存密碼的用戶名或密碼，自動更新修改日期
- **刪除**：永久刪除密碼記錄（有確認提示）
- **列表展示**：顯示所有已儲存密碼的服務名稱、用戶名、最後修改日期

### 4. **導出為 CSV** 📊
- 一鍵導出所有密碼到 CSV 檔案
- 包含欄位：服務名稱、用戶名、密碼、最後修改日期
- 用戶可自選保存位置

### 5. **安全性** 🔒
- **系統級認證**：應用啟動時必須輸入當前系統用戶密碼（Windows LogonUser / Unix PAM），確保只有授權用戶可訪問。
- **金鑰保護**：
  - **Windows**: 使用 DPAPI 將加密金鑰與用戶帳戶綁定。
  - **macOS/Linux**: 使用系統金鑰鏈（Keychain / Secret Service）儲存金鑰。
- **加密存儲**：所有密碼使用 `cryptography.fernet` 對稱加密。
- **記憶體安全**：使用 `SecureString` 處理敏感數據，並對認證狀態進行加密儲存。

## 使用方法

### 啟動應用
1. 執行 `強密碼產生器` 執行檔。
2. 輸入系統用戶密碼進行認證。
3. 認證成功後進入應用主窗口。

### 開發與打包

#### 安裝依賴
```bash
pip install -r requirements.txt
```

#### 執行程式
```bash
python app.py
```

#### 跨平台打包
本專案使用 PyInstaller 進行打包。由於 PyInstaller 不支援交叉編譯，請在目標平台上執行以下指令：

```bash
# 打包為單一執行檔
pyinstaller --clean 強密碼產生器.spec
```

## 安全考量

1. **本地認證**：應用啟動時需輸入系統密碼，防止未授權訪問。
2. **金鑰保護**：金鑰透過系統安全機制（DPAPI/Keyring）保護，無法直接複製到其他電腦使用。
3. **數據庫加密**：所有密碼在數據庫中加密存儲。
4. **剪貼板副本**：複製密碼時仍在剪貼板中，建議使用後立即刪除。
5. **CSV 導出**：導出的 CSV 包含明文密碼，請妥善保管。

## 常見問題

### Q: 忘記了系統登入密碼怎麼辦？
A: 應用依賴系統認證，無法繞過。請通過系統帳戶恢復選項重設密碼。

### Q: 密碼存在哪裡？
A: 密碼加密存儲在數據目錄下的 `passwords.db`。

### Q: 可以在其他電腦上使用嗎？
A: 由於金鑰與硬體/用戶帳戶綁定，無法直接跨電腦共享資料庫。建議使用 CSV 導出功能進行遷移。
